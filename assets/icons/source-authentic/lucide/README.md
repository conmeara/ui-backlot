# Lucide Icons

Curated SVG symbol subsets from `lucide-static` (ISC license,
https://lucide.dev) generated by `tools/build-icon-sprites.mjs` from
`assets/icons/sprite-manifest.json`:

- `claude-symbols.svg` — thin-stroke glyphs for Claude app composer, sidebar,
  topbar, and tool-state rows.
- `browser-symbols.svg` — Chrome-like toolbar and tab-strip glyphs.
- `design-symbols.svg` — Figma-style and Premiere-style tool/panel glyphs.

Lucide's 24px 2px-stroke style is the closest open match to the thin-stroke
iconography in the Claude desktop/web app. Regenerate with:

    node tools/build-icon-sprites.mjs claude browser design
