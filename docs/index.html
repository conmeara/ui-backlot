UI Backlot.

Editable software sets for demo videos. High-fidelity HTML recreations of real apps — scriptable, themeable, and rendered with HyperFrames instead of screen-recorded. Open an application to see every component, variant, dark mode, and wallpaper, then click an install command to copy it.
