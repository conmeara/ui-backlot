# Microsoft Icons

`office-ribbon-symbols.svg` is a legacy, non-manifest file with the first
shared Office ribbon symbols for editable Word/Excel/PowerPoint surfaces. All
six of its symbols (`document-edit-20`, `document-checkmark-20`,
`document-dismiss-20`, `panel-right-16`, `arrow-previous-16`,
`arrow-next-16`) have since been folded into the `office` family in
`assets/icons/sprite-manifest.json` and are generated into `office-symbols.svg`
below, which is now their canonical source. `office-ribbon-symbols.svg` is kept
only for history and is no longer read by any tool or composition.

`office-symbols.svg` is the larger curated ribbon/pane subset generated by
`tools/build-icon-sprites.mjs` from `assets/icons/sprite-manifest.json`
(source: `@fluentui/svg-icons@1.1.331`, MIT). Regenerate with:

    node tools/build-icon-sprites.mjs office

Source basis:

- `@fluentui/react-icons@2.0.330`
- License: MIT
- Local reference checkout: `reference/open-source/fluentui`
- Extraction source during this pass: `npm pack @fluentui/react-icons`

Only the small symbols needed by UI Backlot are tracked here. Do not vendor the
full generated icon package unless a later workflow has a concrete need for it.
