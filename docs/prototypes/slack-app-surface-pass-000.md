# Slack App Surface (Pass 000)

Initial Slack desktop app surface created from real v4.50.143 native app captures and App Store screenshots.

## Reference

- Real Slack desktop app v4.50.143 (native-local capture, 2026-07-13)
- App Store screenshots (9 shots, reference/slack/actual-app/)
- Slack Help Center articles

## Scope

Editable Slack workspace desktop app shell featuring:
- Purple sidebar with workspace avatar and icon rail
- Icon navigation (Home, DMs, Activity, etc.)
- Workspace switcher
- Top search bar
- Message list with reactions and threads
- Open thread panel
- Rich-text compose box

## Notes

- Desktop chrome (titlebar, traffic lights, window border) informed by real Slack app
- Message content and user data are synthetic demo material (workspace uses generic "Acme Robotics")
- Real Slack app is Electron wrapping the web UI; layout matches desktop app at capture
