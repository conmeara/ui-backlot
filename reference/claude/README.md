# Claude Reference Media

This directory is for local reference-only Claude launch videos, screenshots,
and extracted frame studies.

The media files are intentionally ignored by git. They are useful for local
fidelity review, but they should not be redistributed with the UI Backlot source
repo.

## Frame Studies

- `frame-study/sonnet-4-6/`: sampled frames from the 2026-02-17 Sonnet 4.6
  launch reference video.
- `frame-study/code-desktop-redesign/`: sampled frames from the 2026-04-14
  Claude Code desktop redesign reference video, used for the
  `claude-code-desktop` component.
- `frame-study/code-to-figma/`: sampled frames from the 2026-02-18 Claude
  Code-to-Figma reference video, used for the `claude-code-terminal-session`,
  `figma-onboarding-editor`, and `claude-code-figma-workflow` surfaces.
