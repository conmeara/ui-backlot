# Non-Affiliation Note

UI Backlot contains editable recreations of real software interfaces, built to
look as close to the originals as possible for demo videos. The project is an
independent recreation kit and is not affiliated with, endorsed by, or sponsored
by Anthropic, OpenAI, Apple, Microsoft, Figma, Adobe, or any other company whose
product a surface represents. Product names are used descriptively so it is
clear which real workflow a surface recreates; those names and marks belong to
their respective owners.

There are no additional contributor asset rules beyond the one privacy default
in `docs/asset-policy.md` (keep the owner's own logged-in captures out of git).
