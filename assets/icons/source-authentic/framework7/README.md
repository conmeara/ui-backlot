# Framework7 Icons

`macos-symbols.svg` is a curated SVG symbol subset from `framework7-icons`
(MIT, https://github.com/framework7io/framework7-icons), generated by
`tools/build-icon-sprites.mjs` from `assets/icons/sprite-manifest.json`.

Framework7 Icons deliberately follow Apple's SF Symbols visual language, which
makes them the best legally shippable stand-in for Finder sidebar/toolbar
glyphs, menu-bar status icons, and macOS window chrome. Apple's own SF Symbols
may not be redistributed as extracted assets, so we do not vendor them.

Regenerate with:

    node tools/build-icon-sprites.mjs macos
